# LiveKit Three

A beginner-friendly LiveKit voice agent starter repo.

**Python agent** — LiveKit Agents SDK with STT (Deepgram), LLM (GPT-4o), TTS (Inworld), noise cancellation, and inactivity detection. See [python/README.md](python/README.md) for full setup.

**Agent UI** — Frontend for connecting to the agent via LiveKit rooms. Create a `.env.local` with your LiveKit credentials (`LIVEKIT_API_KEY`, `LIVEKIT_API_SECRET`, `LIVEKIT_URL`, `NEXT_PUBLIC_LIVEKIT_URL`), then `npm install && npm run dev`.
